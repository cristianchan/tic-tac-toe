#TIC-TAC-TOE
##Configuration File
To set board size, player names, symbols set this configuration in config.properties file en the next route

`src/main/resources`

```bash

board.size=4
player.number=3

player-one.name=Jose
player-one.symbol=X

player-two.name=Joe
player-two.symbol=O

player-tree.name=CPU
player-tree.symbol=Z

```
